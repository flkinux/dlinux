¿Qué es?

Una distribución Linux ligera enfocada en equipos antiguos. Las aplicaciones están escritas con FLTK 1.4 para ofrecer velocidad, bajo consumo y simplicidad.

Enfoque

Arranque rápido, escritorio minimalista y utilidades básicas: explorador, visor de imágenes, editor de texto y herramientas del sistema.

Estado

Proyecto en desarrollo (sin fecha de salida). Este sitio es una vista preliminar.

Requisitos (estimados)

  • CPU x86_64 antigua
  • 1–2 GB de RAM
  • 8 GB de almacenamiento
